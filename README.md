jQuery.fragmentScroll
=====================

## Example

### Javascript

<pre>$(window).load(function() {
	$("html, body").fragmentScroll();
});</pre>

## Options

<pre>$("html, body").fragmentScroll({
	frame: 'html, body',
	offset: 0,
	speed: 1000,
	easing: 'swing', // http://easings.net
	showFragment: false
});</pre>
